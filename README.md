# js2lua
Javascript to Lua translator

Translates a tiny subset of Javascript to Lua.
School project for ENSTA Bretagne by

Etiene Dalcol

Gurvan Le Bleis

## Dependencies
lua 5.1
luarocks 
lpeg version = 0.10 --> luarocks install lpeg 0.10
lxsh verion >= 0.8.6 --> luarocks install lxsh
